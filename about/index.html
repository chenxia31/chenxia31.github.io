0x01 关于博客

博客自身具有互联网的浪漫属性。希望自己的声音可以在虚拟世界中保留，在另外一份🪐可以被记录下来。

至于为什么选择 Notion，在 2023-11-24 日发生了一个无语的时间。本来是借助 Hexo+Github Page 完成静态博客网站的生成和实现的，其流程是 Hexo 在本地将 markdown 文件转换成为静态网站然后部署到 GitHub 上再通过 GitHub Page 实现访问；但是本人不小心将 GitHub 中的静态网站 git pull 后覆盖了本地的文件夹导致 markdown 消失，导致坚持两年的 blog 烟消云散。

也因此尝试让我跳出舒适圈，选择一个更加新颖的方式构建自己的博客。使用 Notion+nobelium+vercel 实现静态网站部署。

我坚信，分享自己的所见所得，才能让互联网变得更加美好。
